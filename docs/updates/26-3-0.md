# Bedrock Perfected 26.3

::: info
Released on March 1, 2026.
:::

![Version 26.3](/src/updates/26-3-0.webp)

It's been a month since the last major update for Bedrock Perfected — **here's the March update**!

This release is fairly large: it includes many **new tweaks and mechanics** and even a **complete overhaul** of earlier systems.


## Revamped Biome Notifier
Some of you said the previous **Biome Notifier** felt annoying, so this update ships a revamped version.

The new notifier is smaller and uses quieter, improved sounds.

![Biome Notifier](/src/images/new-mechanics/biome-notifier.webp)

Don't worry, the new version is still compatible with custom biomes!

::: info
Shot with **[Biomes and Beyond](https://mcpedl.com/biomes-and-beyond/)** add-on by **iKorbon**.
:::

![Biome Notifier with Biomes and Beyond](/src/images/new-mechanics/bn-bab.webp)


## Settlement Mechanics
Ever wanted to show your city name to other players? Now you can — set a custom radius that notifies players when they enter your settlement.

<video autoplay loop muted playsinline>
  <source src="/src/images/new-mechanics/s/settlement.webm" type="video/webm">
</video>

To do this you need the new **[Scroll of the Hearth](/new-items/scroll-of-the-hearth)** item.

![Scroll of the Hearth](/src/icons/items/scroll-of-the-hearth.png)

Use the item to open this menu.

![Scroll of the Hearth](/src/images/new-mechanics/s/first-menu.webp)

Click Add Settlement then fill out all the form to create your first settlement.

![Scroll of the Hearth](/src/images/new-mechanics/s/add-settlement.webp)

:::details
`Settlement ID` is a unique identifier for your settlement; it cannot match an existing settlement ID.

`Radius X and Radius Z` is the length and width of your settlement.

`Welcome Message` is the small text at the top side of the notification.

`Settlement Name` is the big text at the bottom side of the notification.
:::

## Addon Configurations
Some of you asked for add-on options so you can pick which features to use — now that's available!

You can **choose a theme in the subpacks menu** — Default or Dark.

![Theme Option](/src/images/new-mechanics/subpacks-option.webp)

There's also the **[Addon Configuration item](/new-items/addon-configuration)** to manage which mechanics you want enabled or disabled. This configuration is per-player, not global.

![Addon Mechanics Icon](/src/icons/items/addon-configuration.png)

![Addon Configuration screen](/src/images/new-mechanics/addon-configuration-screen.webp)
