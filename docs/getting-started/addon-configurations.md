# Addon Configurations
You can pick what to enable and what to disable in this addon.

## UI Theme Options
You can pick to use the Default or Dark themed UI in the subpacks options.

![Theme Option](/src/images/new-mechanics/subpacks-option.webp)

## Mechanics Options
![Addon Mechanics Icon](/src/icons/items/addon-configuration.png)

You can use the **[Addon Configuration item](/new-items/addon-configuration)** to manage what mechanics you want to disable or enable. This config is managed per-player and not globally.

![Addon Configuration screen](/src/images/new-mechanics/addon-configuration-screen.webp)
